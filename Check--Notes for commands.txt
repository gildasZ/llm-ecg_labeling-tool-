With the virtual environment (virt env):
	Get pipenv, better than venv:			pip install pipenv
	Create your virt env:				pipenv install
	Or create while installing 1st package:		pipenv install <package-name>
	Activate with: 					'pipenv shell'
	Check if virt env in powershell (shows packages): 'pip freeze'
	For installing packages, better use:		pipenv install <package-name>
	To created a TXT file with exact versions: 	pip freeze > requirements.txt
	To install everything as in the file:		pip install -r requirements.txt
	To update the Pipfile and Pipfile.lock files of the virt env:	pipenv install -r requirements.txt
	Deactivate with: 				'deactivate' or 'exit'

Installing Redis. 
	It is not officially supported by Windows.
	So here are the steps:
		1. 	Enable WSL2 (Windows Subsystem for Linux). 
			WSL2 lets you run Linux binaries natively on Windows.
			For this method to work, you'll need to be running Windows 10 version 2004 and higher or Windows 11.

			Open Windows PowerShell with Administrive privilege and:
				---
					Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
				---
				and you will see:

				Path          :
				Online        : True
				RestartNeeded : False

		2. 	Then:
				---
					start ms-windows-store:
				---
				and it will open Microsoft Store.

		3.	Then search:
				---
					Ubuntu
				---
				and download it and open it.

		4.	For "Enter new UNIX username:", I used:
				---
					gilda
				---

		5.	For the password, I used:
				---
					12345
				---

		6.	Then go on this website to 'Install Redis on Windows':
				---
					https://developer.redis.com/create/windows
				---
